XMAS_LIGHTS_HACK

Reverse Engineering LED Curtains with H-Bridge Drivers.

:: HARDWARE: ESP8266 | :: PROTOCOL: ESPHOME | :: STATUS: PAUSED
// FIG.01: CIRCUIT_TOPOLOGY

01 // CONTEXT

MOTIVATION: BREAKING PROPRIETARY CONTROLLERS

Cheap LED curtains often come with generic, sealed controllers that offer limited modes and no smart home integration. The blinking patterns are preset, non-adjustable, and typically too frantic for ambient lighting. The goal was to bypass the original MCU and inject a WiFi-enabled brain to control the array via Home Assistant.

By analyzing the circuit trace, I discovered the LEDs were driven by reversing polarity on two main lines—a technique used to save copper wire in manufacturing. This necessitated an H-Bridge driver rather than simple MOSFET switching to achieve full control over both "halves" of the LED string.

02 // REVERSE_ENGINEERING

01.

Polarity Analysis

Using a multimeter, I determined that the string was divided into two interleaved circuits. Applying +V to Line A illuminated the odd LEDs, while applying -V (reversing polarity) illuminated the even LEDs. This AC-like behavior is typical of cheap 2-wire LED strings.

02.

The Hack

I desoldered the original controller chip and tapped into the output lines. An L298N H-Bridge motor driver was introduced to handle the polarity switching. An ESP8266 (D1 Mini) sends PWM signals to the H-Bridge input pins to simulate the AC waveform needed to drive both sets of LEDs simultaneously (persistence of vision).

03.

Integration

Flashed ESPHome onto the microcontroller. Configured a custom "Light" component in YAML that exposes brightness and effect controls to Home Assistant. The result is a flicker-free, fully dimmable curtain that responds to automations.

03 // BILL_OF_MATERIALS

[ ESP8266 ] [ H-Bridge Driver ] [ LED Curtain ] [ ESPHome ] [ Soldering Iron ]

04 // CURRENT_STATUS

Project is currently paused pending safety upgrades. The prototype is functional but exposed on a breadboard. Before permanent installation, a custom 3D-printed enclosure is required to shield the high-voltage side of the power supply. Additionally, I plan to build a solder fume extractor before resuming heavy modification work on the remaining strings.
